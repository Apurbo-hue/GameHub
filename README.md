# 🎮 GameHub – A Game Library

GameHub is an online game library where users can explore games, view detailed information, and access download links. The platform allows users to discover popular games and support developers through an engaging interface.

---

## 🌐 Live Links

Netlify: https://gamehub-apurba.netlify.app/

---

## 🎯 Project Purpose

The purpose of GameHub is to build a responsive game discovery platform where users can browse games, view detailed information, and explore download links. The project demonstrates authentication, protected routes, responsive design, and UI interaction using modern React tools.

---

## ✨ Key Features

- Email and Password Authentication using Firebase  
- Google Login Authentication  
- Logout Functionality  
- Protected Game Details Page  
- Popular Games Section sorted by ratings  
- Banner Slider using Swiper  
- Newsletter Section on Homepage  
- User Profile Page  
- Update User Information (Name and Photo URL)  
- Password Validation (Uppercase, Lowercase, Minimum 6 Characters)  
- Forgot Password Feature  
- Custom 404 Page  
- Dynamic Page Title based on route  
- Fully Responsive for Mobile, Tablet, and Desktop
